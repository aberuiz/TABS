# Recode values to and from TDLR

Will submit appropriate codes to TDLR and return plain language values
to user

## Usage

``` r
TABSdecoder(codes, codebook_df = codebook)
```

## Arguments

- codes:

  variable sent to codebook for deciphering

- codebook_df:

  codebook dataframe to check codes against, by default this uses the
  built in \`codebook\`
